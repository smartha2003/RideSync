## How to use?

Access the UI on web:
https://snack.expo.dev/@ylylime/ridesync

OR 

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project on iOS / Android.

```js
 npm run ios // npm run android
 // or
 yarn ios // yarn android
```
